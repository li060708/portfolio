1. 产品概述

1.1 产品定位

认知健康SaaS平台是一款面向体检中心与基层医疗机构的B2B认知健康筛查与管理工具。平台以AD8量表为核心筛查工具，结合语音交互与AI健康咨询，帮助机构高效完成老年人群的认知功能初筛与风险分层管理。

1.2 目标用户

用户角色核心诉求使用场景
体检中心医生快速完成认知筛查，生成标准化报告年度体检中的认知功能评估环节
社区医护人员对辖区老年人进行批量初筛与风险分层社区健康管理、慢病随访
机构管理者查看筛查统计数据，导出报告，管理受测者数据看板、机构汇报
老年受测者通过语音交互完成自测，获取健康建议体检现场或居家自测

1.3 核心价值

对机构：标准化筛查流程 + 数字化管理 + 数据可视化，降低人工成本，提升筛查效率。

对受测者：语音交互降低使用门槛，即时获取风险分层与就医建议。

对行业：填补基层认知健康筛查的数字化缺口，助力"早筛查、早干预"。

2. 用户分析

2.1 核心用户旅程

筛查场景（受测者）：进入自测页面 → 开启语音播报 → 逐题作答（8题/1分钟） → 查看风险等级 → 获取专业建议 → 保存/打印报告。

管理场景（机构）：登录后台 → 查看今日筛查数据 → 按风险等级筛选受测者 → 导出报告/Excel → 跟踪高风险个案。

2.2 痛点分析

痛点现状产品方案
筛查效率低纸质问卷，人工计分，易出错语音引导+自动计分+即时报告
数据难管理Excel或纸质存档，无法追溯趋势结构化数据库+历史记录查询
老年人使用门槛文字阅读困难，操作复杂语音播报+大字体+简化交互
缺乏专业建议只给分数，没有解读和下一步指引四级风险分层+个性化建议

3.2 核心筛查工具：AD8量表

AD8（Ascertain Dementia 8）是国际通用的认知功能初筛工具，共8个"是/否"问题，评估记忆、执行功能、方向感等维度。约1分钟完成，适用于大规模初筛场景。

4.2 详细功能描述

4.2.1 AD8语音自测（P0）

入口：机构提供二维码或链接，受测者扫码进入。

交互流程：语音播报题目 → 用户选择"是/否" → 自动进入下一题（可关闭自动连播） → 8题完成后展示结果。

语速设置：慢/正常/快三档，默认正常。

断点续答：中途退出自动保存进度，下次进入可继续。

4.2.2 四级风险分层（P0）

等级AD8得分判定建议动作
🟢 低风险0-1正常范围每年常规复查
🟡 需关注2-3早期变化迹象3-6个月复查，开始生活方式干预
🟠 中高危4-5可能存在认知损害建议前往神经内科/记忆门诊评估
🔴 高风险6-8高度提示认知障碍尽快全面检查，家属加强照护

4.2.3 AI健康咨询（P1）

基于大语言模型的MCI健康咨询智能体，覆盖知识问答、生活方式干预、情绪安抚三大场景。已建立21条评测用例的测试体系，完成从Prompt V1.0到V2.0的迭代优化。

5. 非功能需求

维度要求
性能自测页面加载 ≤ 2秒；语音播报延迟 ≤ 500ms
兼容性支持iOS 13+、Android 8+、主流微信版本内置浏览器
可用性字体≥16px，按钮≥44px触控区域，语音引导覆盖所有操作
安全性用户健康数据加密存储，传输层TLS，RBAC权限控制
合规筛查结果附带"不构成医疗诊断"声明；不存储未脱敏的个人身份信息

6. 数据与指标体系

指标类型指标用途
业务指标筛查完成率、高风险检出率评估筛查工具的有效性

体验指标答题完成率（8题全答完/开始答题）衡量交互流程的流畅度
